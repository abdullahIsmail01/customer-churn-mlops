# Customer Churn Prediction using MLOps

## Project Overview

This project implements a complete MLOps pipeline for predicting customer churn using the IBM Telco Customer Churn dataset.

The project includes:

- Data versioning
- Data preprocessing
- Feature engineering
- Machine learning model training
- Model evaluation
- MLflow experiment tracking
- Docker deployment
